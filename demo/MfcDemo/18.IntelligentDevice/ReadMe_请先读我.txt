【Demo功能】
1、Demo介绍SDK初始化,登陆设备, 登出设备, 订阅智能事件, 接收智能事件报警, 解析智能事件信息, 停止智能事件,  录像查询下载
2、Demo演示的智能事件包括：区域入侵， 绊线入侵; 支持检测识别触发报警事件的物体是人或者是车.

【注意事项】
1、编译环境为VS2005。
2、如需运行demo的exe文件，请将开发包中【库文件】里的全部dll文件拷贝至该工程目录下的bin/x86release或者bin/x64release下。
3、如把库文件放入程序生成的目录中，运行有问题，请到大华官网下载最新的网络SDK版本：http://www.dahuatech.com/index.php/service/downloadlists/836.html 替换程序中的库文件。

【Demo Features】
1.Demo SDK initialization,login device,logout device, subscribe to intelligent event information, receive intelligent event information, parse event information, video query and download. 
2.Demo a part of intelligent event include Tripwire, Intrusion; Supporting detection and recognition of objects triggering alarm events is human or vehicle.

1.Complier for Demo is VS2005.
